# Проект 22. Знакомство с Django

## Описание:

В этом проекте будет реализован вариант интернет-магазина на движке фреймворка Django.

## Установка:

1. Клонируйте репозиторий:

```
git clone https://github.com/a-babushkin/Django_Basics.git
```

2. Установите зависимости:

```
poetry install
```

## Документация:

Для работы с этим приложением надо сначала запустить локальный вэб-сервер командой: ``` python manage.py runserver ```
и перейти по ссылке http://127.0.0.1:8000/. Откроется главная страница приложения. Далее можно переключаться между
Главной страницей и страницей контакты. Так же можно заполнить контактную форму и отправить на сервер, в результате
получите отправленные данные на сервер в броузере ответом от него.

## Доработки:

Популярность магазина значительно выросла. В связи с этим продакт-менеджер предложил добавить возможность пользователям
заполнять карточки продуктов и при этом расширить функционал веб-приложения.
